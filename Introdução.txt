A Comissão de Educação da Câmara dos Deputados aprovou projeto que institui a Política Nacional de Estímulo ao Uso de Jogos Eletrônicos na Educação Básica (PNJE), que tem o objetivo de aprimorar a aprendizagem por meio do uso de jogos eletrônicos.

Fonte: Agência Câmara de Notícias 17/09/2021 - 15:35
  
 Colocar no jogo: Vida extra

 Análise de concorrencia:
 Pontos Positivos:  Roda em qualquer computador, obtemos uma forma de ranking por tempo e pontuação, software desenvolvido especialmente para educação nas escolas 


Funcionais: 
            F1 - O sistema deve realizar cadasto de usuario 
            F2 - O sistema deverá enviar um email automatico confirmando o cadastro do usuario.
            F3 - O software devera ter um excelente desempenho. 
            
Não Funcionais: NF 2.1 - O usuario deverá se cadastrar usando seu endereço de e-mail.
                NF 2.2 - O sistema deverá utilizar uma base de dados para o armazenamento de informações
                NF 2.3 - 


                NF 1.1 - Usuario devera realizar o cadastro utilizando o seu endereço de e-mail
                NF 1.2 - O cadastro no ranking devera ter no maximo 10 caracteres 
                NF 1.3 - A senha do usuario devera ser apenas letras e numeros.

 
                NF 3.1 - O software devera abrir em 1 segundo
                NF 3.2 - O jogo devera rodar em um sistema windows
